# 🛵 Nepal Delivery - Backend API

A robust and secure backend REST API built for a food delivery ecosystem in Nepal. This project utilizes the MERN stack (Node.js, Express.js, MongoDB Atlas) to handle restaurant registrations, data storage, and retrieval.



## 🌟 Key Features
* **Cloud Database Integration:** Connected to MongoDB Atlas for scalable data storage.
* **DNS Bypass Mechanism:** Custom implementation to bypass local network/ISP restrictions for database connectivity.
* **Secure Configuration:** Environment variables (dotenv) used to protect sensitive credentials.
* **Professional Debugging:** Includes utility scripts for network diagnostics.

## 🚀 Tech Stack
* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB Atlas (Cloud)
* **ORM:** Mongoose
* **Security:** Dotenv & CORS
